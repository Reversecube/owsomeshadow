# owsomeshadow

### Installation

`npm i owsomeshadow --save`

Then .....

```
import {owsomeshadow} from 'owsomeshadow';

owsomeshadow({
    shadow_type : 'soft',
    padding : false
});
```

### Options

owsomeshadow supports 2 options,both of wich are optional:

- _shadow_type_ - _hard | soft_ (Defaults to soft)
- _padding_ - _boolean_ (Defaults to false)
